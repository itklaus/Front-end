# Front-end.
Author: [itklaus](https://github.com/itklaus)

## How to Start:

You must have:
1. nodejs
2. npm
3. gulp

Next:
1. Download Front-end from GitHub;
2. Install Node Modules: npm i;
3. Run the template: gulp.

## What is included?
* BrowserSync
- compile jade and stylus
+ compression styles, js and images
